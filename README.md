[# SyousetuReader](https://uni928.github.io/SyousetuReader/)

小説をローカルで読めるようにするためのサイトです。

自動ではありません。小説をコピーして貼り付ける作業が必要なので、作業量は膨大化と思います。

[# SyousetuReader リアルタイムセーブ版](https://uni928.github.io/SyousetuReader/index2.html)

リアルタイムセーブがあるバージョンです。

ブラウザが落ちる事でデータが飛ぶのが嫌な方はこちらをお試しください。

[# SyousetuReader 出力サンプル](https://uni928.github.io/SyousetuReader/index5.html)

ChatGPTで出力したマインクラフト小説の出力サンプルです。

音声読み上げの快適さなどを体感して下さい。

読み上げ速度は2.0を推奨します。

画面オンなど、スリープ防止アプリの併用が必要かもしれません。

[# SyousetuReader ポータルサイト作成用](https://uni928.github.io/SyousetuReader/index6.html)

各小節へのリンクを配置したサイトを出力するためのサイトです。小説が増えた場合にご利用下さい。
